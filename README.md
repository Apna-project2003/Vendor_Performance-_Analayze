# Vendor Performance & Profitability Dashboard

## Project Overview
This Power BI dashboard analyzes vendor sales, profitability, stock turnover, freight cost, and operational efficiency. The dashboard helps identify top-performing vendors, low-performing vendors, and key business insights.

## Tools Used
- Power BI
- DAX
- Data Modeling
- CSV Data Sources

## Dashboard Pages

### 1. Overview
- Total Sales
- Gross Profit
- Total Vendors
- Profit Margin
- Top Vendor Revenue Analysis

### 2. Vendor Analysis
- Top Profit Margin Vendors
- Bottom Profit Margin Vendors
- Vendor Profitability Distribution
- Sales vs Gross Profit Analysis

### 3. Operations
- Stock Turnover Analysis
- Freight Cost Analysis
- Purchase Price vs Sales Revenue

### 4. Insights
- Operational Efficiency Insights
- Vendor Performance Insights

## Key Features
- Interactive Navigation Menu
- Dynamic Filters
- KPI Cards
- Bar Charts

- Vendor Performance Analysis
