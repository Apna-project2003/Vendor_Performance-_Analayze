# 📊 Vendor Performance & Profitability Dashboard

## 📌 Project Overview

This project analyzes vendor sales performance, profitability, stock turnover, purchase contribution, and operational efficiency using SQL, Python, Excel, and Power BI.

The dashboard helps business stakeholders identify top-performing vendors, low-performing vendors, profit trends, and operational improvement opportunities.

---

## 🎯 Business Objectives

- Identify Top & Bottom Vendors
- Analyze Gross Profit Performance
- Track Vendor Profitability
- Measure Stock Turnover Efficiency
- Compare Sales vs Purchase Metrics
- Support Business Decision Making

---

## 🛠️ Tech Stack

| Tool | Usage |
|--------|---------|
| SQL | Data Extraction & Querying |
| Python | Data Cleaning & Preprocessing |
| Excel | Data Validation & Analysis |
| Power BI | Dashboard Development |
| DAX | KPI Calculations |

## 📂 Dataset

Vendor Sales Summary Dataset

Columns Include:

- VendorName

- Purchase Price
- Freight Cost

- Profit Margin

- Top vendors contribute majority of revenue.
- Several vendors show negative profit margins.
